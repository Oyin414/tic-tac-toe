# About TIC TAC TOE
This TiC TAC TOE game is a lighthearted two player game that users can play with either their friends or families. In this game Player One is crosses and player Two is noughts, the first player to get three noughts or three crosses in a row wins the game . Players can play the game however many times as they'd like and there's a reset button that allows the grid to be reset .

**Link to project:** https://tic-ta-c.netlify.app/

Video WalkThrough : https://youtube.com/shorts/wm1IDlydfT4?feature=share

## How It's Made:

**Tech used:** HTML, CSS, JavaScript

This web application was made using Object Oriented Programming, this project in particular made use of one of the key concepts in Object Oriented Programming ,Encapsulation, through the usage of this key concept it allows the code to have better maintainability, readability and usability and it helps to protect sensitive data from unauthorized access.


## Lessons Learned:
 
During this project my main goal was to have as little global code as possible which required me to think carefully about where each piece of code went . I had to go rewrite sections of code that contained too much repetition and it took me a bit of time rendering the contents of my array onto the page and making sure that when players interacted with the grid , the player values were recorded and added to my array .
